## 用法
用这个启动爬虫项目 `Crawler.start("https://portal.sina.com.hk/")`，每次爬取停3s防止封ip。
使用数据库存储数据，项目中建 `db.config.json` 文件。文件格式如下：
```json
{
  "jdbc": "xxx",
  "user": "xxx",
  "password": "xxx"
}
```
爬取可以断点续传。
